# Ecoway Men Products

متجر إلكتروني عربي RTL لمنتجات إيكواي الرجالية — HTML/CSS/JS خالص، متوافق مع GitHub Pages.

## الصفحات

- `index.html` — الرئيسية
- `products.html` — المنتجات (بحث + فلاتر)
- `product.html` — تفاصيل المنتج
- `cart.html` — سلة التسوق (localStorage)
- `checkout.html` — إتمام الطلب (POST → Google Apps Script)

## الإعداد

### 1. Google Apps Script

عدّل `orderEndpoint` في `assets/js/config.js` برابط الـ Web App الخاص بك.

### 2. GitHub Pages

الإعدادات → Pages → Source: `main` branch → `/ (root)`

الرابط: `https://ecoway-menproducts.github.io/ecoway-men-products/`
