# websites-exp — مرجع دراسة مواقع E-Commerce المشابهة

> **الغرض:** توثيق تحليل مواقع تجارة إلكترونية مرجعية (عطور، أزياء، marketplaces، RTL) لاستخدامه أثناء تصميم وبناء الموقع الجديد.  
> **آخر تحديث:** 2026-07-01  
> **الحالة:** قيد التوسع — **5 مواقع** مُحلَّلة (Yfouh + SUTRA + Men's Club + Joom + Ecoway).

---

## كيفية استخدام هذا الملف

1. قبل أي قرار تصميمي → راجع قسم **«ما نتجنبه»** و**«توصيات مشروعنا»**.
2. عند مشاركة موقع جديد → يُضاف قسم جديد بنفس القالب أدناه.
3. عند بناء الموقع → راجع **لوحة الألوان المرجعية** و**قائمة التحقق UX**.

### 1.5 التسعير (تحليل تقني)

تم فحص **149 منتج / 596 variant** عبر Shopify JSON API:

| النمط | العدد | الوصف |
|-------|-------|-------|
| **أسعار مقلوبة** | 447 variant | `price` > `compare_at_price` — العميل يدفع **أكثر** والواجهة تعرض «تخفيض» |
| **أسعار صحيحة** | 149 variant | غالباً حجم 10ml فقط |
| **بدون compare_at** | 0 | |

**مثال (30ml):**
```
price:          290.00  ← ما يُدفع فعلياً
compare_at:     199.00  ← يُعرض كـ "السعر الأصلي"
الواجهة:        ~~199~~ → 290 + شارة Sale  ← عكس المنطق
```

**نمط تسعير موحّد مشبوه:**
- معظم المنتجات: 290 / 199 بغض النظر عن الماركة (Dior, Chanel, Tom Ford...)
- يقلل مصداقية متجر منتجات فاخرة

## قائمة تحقق UX قبل الإطلاق

### تسعير وسلة
- [ ] `price` < `compare_at_price` دائماً عند وجود تخفيض
- [ ] السعر في السلة = السعر المعروض على الصفحة
- [ ] اختبار شراء افتراضي لـ 3 منتجات × 3 أحجام
- [ ] رمز عملة واحد في كل الموقع

### لغة ومحتوى
- [ ] 100% عربي في الواجهة (أو ثنائي لغة متعمد ومتسق)
- [ ] لا نصوص Shopify افتراضية
- [ ] مراجعة إملائية لكل CTA
- [ ] سياسات مترجمة ومخصصة للسوق المصري

### ثقة ومحتوى
- [ ] صفحة «من نحن» ← **من SUTRA**
- [ ] صفحة «اتصل بنا» (واتساب + بريد + نموذج)
- [ ] صفحة فروع (إن وُجدت) ← **من SUTRA**
- [ ] صفحة شحن وتوصيل
- [ ] FAQ ← **من SUTRA**
- [ ] جدول مقاسات (للملابس) ← **من SUTRA**
- [ ] مراجعات عملاء ← **من SUTRA**
- [ ] **وصف مخصص لكل نوع منتج** — لا قالب ملابس للعطور ← **من Men's Club**
- [ ] شارات ثقة على الرئيسية ← **من Men's Club**
- [ ] طرق الدفع على صفحة المنتج ← **من Men's Club**
- [ ] صور منتجات احترافية — لا AI بأسماء واضحة
- [ ] إخفاء/تقليل المنتجات غير المتوفرة في الصفحة الرئيسية ← **درس SUTRA**

### موبايل
- [ ] شريط تنقل سفلي عربي (رئيسية / قائمة / بحث / متجر / سلة) ← **من SUTRA**
- [ ] Quick View للمعاينة السريعة ← **من SUTRA**
- [ ] نظام مراجعات قوي ← **من Joom**
- [ ] تتبع شحنة شفاف ← **من Joom**
- [ ] توصيات منتجات ذات صلة (خوارزمية) ← **من Joom**
- [ ] Mobile-first في التصميم ← **من Joom**

### SEO
- [ ] عنوان ووصف فريد لكل صفحة
- [ ] og:image مخصص
- [ ] Structured data (Product, Organization)
- [ ] URLs عربية أو وصفية

### أداء
- [ ] خطان كحد أقصى
- [ ] صور WebP مضغوطة
- [ ] اختبار سرعة (LCP < 2.5s)
